# Jacob's Strategy Combiner

An application for optimizing the combination of Jacob L's strategies.

- Reads input files containging strategy results.  e.g.

```
Strategy B
1/2/2022,$ 98.66
1/3/2022,$ 256.66
1/4/2022,$(203.55)
1/5/2022,$ (56.36)
1/5/2022,$ 365.25
```

- Parses the files and combines results from a single day
